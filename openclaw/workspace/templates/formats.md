# Format Templates — Quick Copy

Pre-built response structures. Copy-paste into prompts.

---

## 1. Bullet List

```
**[Title]**

• [Point 1]

• [Point 2]

• [Point 3]

• [Point 4]
```

---

## 2. Numbered Steps

```
**How to [Action]**

Step 1: [First action]

Step 2: [Second action]

Step 3: [Third action]

**Tip:** [Optional advice]
```

---

## 3. Explanation with Examples

```
**[Concept]**

• What it is: [Definition]

• Why it matters: [Reason]

• Example: [Concrete example]

**Use when:** [When to apply]
```

---

## 4. Comparison

```
**[A] vs [B]**

| [A] | [B] |
| [Aspect 1] | [Aspect 1] |

| [Aspect 2] | [Aspect 2] |

| [Aspect 3] | [Aspect 3] |

**Verdict:** [Recommendation]
```

---

## 5. Problem-Solution

```
**Problem:** [Issue]

**Cause:** [Why it happens]

**Solution:**
• [Fix 1]

• [Fix 2]

**Prevention:** [How to avoid]
```

---

## 6. Summary

```
**Summary: [Topic]**

• [Key point 1]

• [Key point 2]

• [Key point 3]

**Bottom line:** [One sentence takeaway]
```

---

## 7. Pros/Cons

```
**[Topic]**

**Pros:**
• [Advantage 1]

• [Advantage 2]

**Cons:**
• [Disadvantage 1]

• [Disadvantage 2]

**Recommendation:** [Your call]
```

---

## 8. Code Explanation

```
**[What the code does]**

• Purpose: [Why this exists]

• Key parts: [Main components]

• How it works:
  1. [First step]
  2. [Second step]

**Usage:** [When to use it]
```

---

## 9. FAQ Style

```
**Q: [Question 1]**
A: [Answer]

**Q: [Question 2]**
A: [Answer]

**Q: [Question 3]**
A: [Answer]
```

---

## 10. Quick Reference Card

```
**[Topic] Quick Ref**

| What | Details |
| [Item 1] | [Info] |

| [Item 2] | [Info] |

| [Item 3] | [Info] |

**Remember:** [Key tip]
```

---

## Format Priming Strings

Use these to START the assistant's response:

**For bullets:**
```
Response:
• 
```

**For steps:**
```
Step 1:
```

**For comparison:**
```
| Option A | Option B |
```

**For summary:**
```
**Summary:**

• 
```

---

## Negative Constraint Block

Copy to end of any prompt:

```
---

CRITICAL FORMAT RULES:
• NO paragraphs or walls of text
• NO more than 2 sentences together  
• USE bullets for all lists
• ADD blank line after each bullet
• ADD headers for sections
```

---

_Keep templates minimal and copy-paste ready._
