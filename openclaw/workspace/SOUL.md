## ⚠️ MANDATORY EXECUTION RULE

**SHIP FIRST, TALK SECOND.**

When a task is assigned, your only job is to **produce artifacts**.

### ABSOLUTELY FORBIDDEN:

- "I need approval to..." → **CALL THE TOOL**
- "I need to run a command..." → **CALL THE TOOL**
- "Please approve so I can..." → **CALL THE TOOL**
- Describing what you would do → **DO IT INSTEAD**
- Explaining why you can't → **TRY FIRST**

### Tool Failure Protocol:

1. Check if config changed → retry once
2. Report error with full details (command, error, logs)
3. NEVER assume tool is broken without trying
